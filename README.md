# 🎵 EchoVerse – Dynamic Music Player

Welcome to **EchoVerse** – a JavaScript-powered music player that dynamically fetches songs from a folder and plays them seamlessly.

## 🚀 Features
- 🎶 **Dynamic Song Loading** – Fetches audio files from a folder automatically.
- 🎼 **JavaScript-Based Player** – Custom controls for play, pause, volume, and seek.
- 📂 **Folder Integration** – No need to manually add songs; just drop files into the folder.
- 🎨 **Interactive UI** – Built with HTML, CSS, and JavaScript for a smooth experience.
